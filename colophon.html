Colophon


This website is inspired by forum and Wikipedia styles of the mid-2000s, where I fell in love with the internet.

I am using the open source typeface Latitude, created by Ange Degheest, which I found at Velvetyne. In the main text I am using Helvetica.

To make this website, I use the free version of BBEdit or VSCode to write the HTML and CSS. I host on Cloudflare, and will move files onto GitHub soon(!).
